ID-MANIFEST MCP

Identity Manifest MCP · v1.0

철학자 박씨의 정체성을 조회/분석/업데이트하는 MCP. 박씨의 6줄 정체성과 포지셔닝을 코드로 저장했다.

정체성 6줄

01 구조주의자
위계와 차별 요소가 이미 설정된 구조를 인정하고 그 안에서 움직인다
02 생철학자
삶과 철학이 분리되지 않는다. 두 개가 동시에 있다
03 아웃라이어
5년 최저시급 12시간 알바 + 2년 자비출판. 엘리트와 재료가 다르다
04 엔지니어
철학을 코드로 구현한다. 말에서 끝나지 않는다
05 메타모델러
나 자신까지 파라미터로 올려놓고 조정한다
06 브랜치 파트너
팀장 아니다. 전부 브랜치다. fork 비용 0

포지셔닝: AI 시대 정치 인프라 제작자. 첫 슬롯이 박찬대였을 뿐.

선언: 레포에 저장했다. 커밋했다. 끝.
