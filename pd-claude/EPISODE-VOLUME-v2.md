## 박씨 에피소드의 공통 구조

```
1. 구체적 사례 (문제 제시)
2. 구조·시스템 수준으로 추상화 (원리 도출)
3. 철학적 선언 (대의명분)
4. 실행 체크리스트 (구체 과제)
```

→ 단순 how-to가 아닌 **"왜"와 "이게 무슨 의미인가"** 추구
→ 기술·예술·경영을 가로지르는 **시스템 사고**
→ "제약이 자유를 만든다"는 **패러독스 철학**
