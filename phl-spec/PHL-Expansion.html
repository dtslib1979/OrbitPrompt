PHL-Expansion

모듈 확장 · 견고화 Generator

대상 모듈을 입력하면 Claude Code에 바로 붙여넣을 PHL-Expansion 실행 프롬프트를 생성한다.

적용 범위 선택

추가 컨텍스트 (선택)

생성된 프롬프트 — Claude Code에 붙여넣기

Name:    PHL-Expansion
Intent:  현재 모듈을 확장 가능하고 견고하게 만든다

Procedure:
  Step 1: 현재 상태 파악 (파일 구조, 에러/로깅 패턴, 테스트, 의존성)
  Step 2: 에러 처리 강화 (try/catch, 컨텍스트 포함 메시지, Result 타입)
  Step 3: 로깅 추가 (진입/종료/에러/핵심 분기)
  Step 4: 인터페이스 정리 (SRP, 매개변수 3개 초과→객체)
  Step 5: 테스트 추가 (happy path, error path, 경계값)

Commit Format:
  PHL-Expansion: <module> - <summary>
  - 에러 처리: [변경 내용]
  - 로깅: [변경 내용]
  - 인터페이스: [변경 내용]
  - 테스트: [추가/수정 내용]

Rollback: 테스트 실패 시 git revert
