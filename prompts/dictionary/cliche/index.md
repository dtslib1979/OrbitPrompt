# 💬 CLICHÉ Thesaurus

> **클리셰 식별 + 대안 제시 사전. 편집술 "표절도 창조도 아닌 편집" 정합.**

## 정의

박씨 편집술 헌법: *"있는 것을 내 방식으로 재조합한다."* 클리셰는 편집의 1차 원료. 식별 → 분해 → 재조합 → 새 의미.

## 7개 SEED (Garage 보존)

기존 CLICHÉ DICTIONARY 7영상이 시작점. Garage 플레이리스트 안에 있음.

## YouTube 연동

- **플레이리스트**: 💬 CLICHÉ Thesaurus (8번)
- **원본**: Garage 안 CLICHÉ 7영상
- **확장**: 한국어 클리셰 100개 영상화 목표

## Generator

`cliche-breaker.md` — 신규 OrbitPrompt Generator:
1. 입력: 텍스트 또는 영상 스크립트
2. 추출: 클리셰 표현 N개 자동 식별
3. 분해: 각 클리셰 7축 분석 (의미/감정/시대성/사용맥락/대체가능성/패러디가능성/생산성)
4. 재조합: 대안 표현 또는 패러디 N개 제시

## 파일

- `cliche-thesaurus.json` — 클리셰 데이터베이스 (한국어 + 영어, 펜딩)
- `cliche-breaker.md` — Generator 정의 (펜딩)
