# ⚡ SYMBOL Database

> **상징/아이콘 체계화. Identity 7축 정합. 28레포 시각 정체성 SSOT.**

## 정의

상징은 의미의 압축. 1개 아이콘 = 1 문장 이상. 박씨 28레포 각자의 상징 + 박씨 페르소나(블로거/뮤지션/비주얼라이저/테크니션/필로소퍼) 상징 + 6 스킬셋 상징 정합.

## YouTube 연동

## Generator

`symbol-encoder.md` — 신규 Generator:
1. 입력: 텍스트/개념/레포명
2. 추출: 핵심 상징 1개 (이모지/아이콘/SVG)
3. 7축 정합 (Identity model 정합)
4. 출력: 상징 + 의미 설명 + 사용 맥락
