Physical Output Format

🧾 Printer Format

ESP32 + 감열식 프린터로 철학·출판·암기 콘텐츠를 물성화하는 출력 포맷.
576px × 무제한 롤지 — 디지털로 만들어서 아날로그로 소비한다.

📐 기술 사양

항목값비고

📋 서식 분류

📜 철학·명상
7축 분석 결과, 철학자 일지, 명상 카드. 좌: 질문 / 우: 해설 2컬럼
📖 출판물 암기장
챕터 요약 → 핵심개념 → 빈칸 문제. 접으면 암기카드, 펼치면 정답
🗂️ 단어장
앞면 단어 / 뒷면 뜻. 세로 롤지에 20개 단위로 잘라서 사용
📋 처방전
철학/학습 처방 포맷. 문제 진단 → 처방 항목 → 복용법(실행 계획)
📊 정산표
커밋=전표의 물성화. PHL 토큰 실행 결과를 정산서로 출력
🔖 인용구 카드
철학자 한 줄, 출처, 박씨 코멘트. 지갑에 넣고 다니는 카드

🔧 파이프라인

# 음성 → 철학/출판 콘텐츠 → 576px 출력 # # 1. 음성 입력 (핸드폰) # 2. STT → Claude Code # 3. 서식 선택 (철학카드/암기장/처방전/정산표) # 4. ESC/POS 바이트 생성 # 5. WiFi → ESP32 → UART → 프린터 # 6. 576px × 필요한 길이만큼 출력 # 7. 케이스 키트 장착 → 아날로그 태블릿 # # 명령어 예시: # python3 printer_format.py --template philosophy_card --text "7축 분석 결과..." # python3 printer_format.py --template memorization --file chapter3.md
