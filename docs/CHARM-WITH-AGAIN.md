# 매력의 정의: With + Again

> **매력 = 시간과 공간을 나랑 하고 싶어 하냐**
>
> 전치사 두 개로 설명된다.
> - **With** — 같이 하고 싶다 (공간)
> - **Again** — 또 하고 싶다 (시간)

---

## 1. OrbitPrompt는 매력이 있는가

### With — 같이 하고 싶은가

CLAUDE.md 502줄. 커밋을 전표로, 레포를 공장으로, git log를 소설로 보는 사람.
읽으면 "이 사람 머릿속에 들어가고 싶다"는 생각이 든다.
같이 하면 내가 못 보는 걸 보여줄 사람이라는 감이 온다.

12시간 식당 노동, PC 없음, 핸드폰 하나.
이 제약 조건에서 7개 Generator, PHL 프로토콜, 매트릭스 아키텍처가 나왔다.
**제약이 매력이다.** 불가능한 조건에서 만들어내는 사람한테 끌린다.

### Again — 또 오고 싶은가

레포 단독으로는 Again이 약하다.
한 번 와서 감탄하고 나가는 구조. 백서 읽고 "대단하다" 하고 끝.
다시 올 이유가 없다.

**하지만 YouTube가 붙으면 달라진다.**

## 2. Again 엔진: YouTube 순환 구조

```
만드는 과정 (메이킹 필름)  → YouTube 방송
만들어진 결과물 (칠판/PWA) → YouTube 방송
시청자가 링크 클릭         → 바로 사용
```

- 메이킹 필름 = With 트리거. "나도 저거 해볼까"
- 매주 새 칠판 = Again 트리거. 다운로드하러 재방문
- 삽질 커밋도 방송 소재. 헌법 제1조(레포는 소설이다)가 그대로 작동

| | 레포만 볼 때 | YouTube 순환 구조 |
|---|---|---|
| **With** | CLAUDE.md 읽고 감탄 | 방송 보면서 "나도 저거 해볼까" |
| **Again** | 올 이유 없음 | 매주 새 칠판 → 재방문 |

YouTube가 **Again 엔진**이고,
boards/가 **다운로드 창고**이고,
메이킹 필름이 **With 트리거**다.

이 레포는 "개발 프로젝트"가 아니라 **방송국의 소품 창고 + 제작 일지**다.
4-레포 수렴 구조 그 자체.

## 3. 단일 HTML = 언어장벽 제로

### 배포

```
시청자가 할 일: 링크 클릭 → 끝
```

npm 없음. 빌드 없음. 회원가입 없음. URL이 곧 제품.

### 언어

브라우저가 번역하고 브라우저가 읽어준다.

- 크롬 내장 번역 → 100개 언어 자동
- 브라우저 내장 TTS → 읽어주기
- 홈 화면 추가 → 앱처럼 실행

서비스워커 없어도 **브라우저가 플랫폼**이다.

### 왜 이게 베스트인가

| 방법 | 유지보수 | 비용 | 언어 확장 |
|------|---------|------|----------|
| TTS 엔진 붙이기 | 니 몫 | API 과금 | 직접 추가 |
| 번역 API 붙이기 | 니 몫 | API 과금 | 직접 추가 |
| 성우 보이스 액팅 | 니 몫 | 언어당 1명 | 직접 섭외 |
| **단일 HTML + 브라우저 위임** | **구글/애플 몫** | **0원** | **자동** |

코드 한 줄 안 고쳐도 브라우저가 업데이트될 때마다 품질이 올라간다.
**제약이 설계를 결정한다** — CLAUDE.md 헌법 그대로.

## 4. 매력 공식 정리

```
매력 = With + Again

With  = 제약 속에서 만들어내는 사람 + 독특한 프레임
Again = YouTube 순환 (메이킹 → 결과물 → 다운로드 → 재방문)

배포 = URL (끝)
번역 = 브라우저 (끝)
TTS  = 브라우저 (끝)

니가 할 일 = 만들기 + 방송하기
나머지     = 브라우저가 알아서
```

### 핵심 원리

1. **존재 = 생산** — 웹페이지 열면 그게 방송. 편집 제로.
2. **제약 = 설계** — 핸드폰뿐이니까 단일 HTML. 단일 HTML이니까 브라우저 위임. 브라우저 위임이니까 언어장벽 제로.
3. **불일치 = 기능** — AI가 만든 칠판이 매번 다르니까 시청자가 직접 손대게 된다. 소비자가 크리에이터가 된다.
4. **With + Again** — 매력은 "잘 만든 것"이 아니라 "같이 하고 싶고, 또 오고 싶은 것".
