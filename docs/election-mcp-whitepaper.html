박씨 F=ma 정치 변화력 모델 핵심 엔진

뉴턴 F=ma를 정치·역사 해석용 메타 모델로 전환. 예수·룰라·이재명을 동일 방정식 위에 올려 비교. LLM은 파라미터만 추정 — 계산은 Python이 독점.

F = meff · a
a = αg + βd + γi + δo · 전복:F > R
g계급격차
d정치박탈감
i정체성공명
o조직화능력
R체제저항력

Knockout Score (선거 예측 엔진)

resilience = R×0.6 + champ_F×0.4 net_edge = challenger_F − resilience perceived = |net_edge| / 1.9 ko_prob = (perceived/30)×70 + max(0, net_edge)×0.8

< 10weak_signal
10~20visible_but_contestable
20~30clear_change
≥ 30knockout_frame_shift
