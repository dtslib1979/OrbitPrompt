# OrbitPrompt — 정체성 선언

> **버전:** 2026-06-09  
> **장르:** 개인 사고 운영체제  
> **커널:** Φ7 드라이버 — 7축 사고 프레임워크  

---

## 1. 이 레포지토리는 무엇인가

**OrbitPrompt = Φ7을 커널로 하는 개인 사고 운영체제.**

일반적인 OS가 프로세스와 파일을 관리하듯, 이 레포는 **네 사고방식과 판단을 관리한다.** 아래 6개 레이어로 구성되어 있고, 전부 연결되어 있다.

| 레이어 | 역할 | OrbitPrompt에서의 구현 | 상태 |
|:------:|------|----------------------|:----:|
| 🧠 **커널** | 사고 프레임워크 | Φ7 드라이버 — 7축 철학 | ✅ |
| ⚙️ **디바이스 드라이버** | 도메인 분석 모듈 | MCP 10개 (정치/축구/달러/철학/미디어…) | ✅ |
| 📝 **파일시스템** | 철학 문서·백서·로그 | docs/ 43개 + boards/ 15개 | ✅ |
| 🖥️ **데스크탑** | 전시·실행 인터페이스 | 갤러리 (SVG 노드맵) | ✅ |
| 🌐 **네트워크 스택** | 외부 유통 채널 | YouTube 3개 + Telegram | ✅ |
| 🔒 **보안 커널** | 편향 검증·자기검증 | Media MCP engine.py (확증편향 방지 + Flux + review) | ✅ |

---

## 2. 이 레포지토리가 아닌 것

| 오해 | 실제로는 |
|------|---------|
| "철학 블로그" | ❌ 블로그가 아니다. 철학을 **실행 가능한 MCP로 컴파일**하는 레포다 |
| "개발 문서" | ❌ 개발 문서가 아니다. MCP는 부산물이고, **핵심은 사고방식 자체**다 |
| "AI 프로젝트" | ❌ AI 프로젝트가 아니다. AI는 도구일 뿐, **이 레포의 주인은 네 사고방식**이다 |
| "유튜브 채널" | ❌ 유튜브는 출력 채널 중 하나일 뿐. **레포 자체가 본체**다 |
| "지식 관리 시스템" | ❌ 지식 관리도 부수적 기능. **본질은 '생각을 운영하는 OS'** 다 |

---

## 3. 왜 OS여야 하는가

| OS의 특성 | OrbitPrompt의 대응 |
|-----------|-------------------|
| 커널이 모든 걸 통제 | Φ7 드라이버 = 모든 MCP의 기반 철학 |
| 디바이스 드라이버로 확장 | MCP = 각 도메인(정치/축구/달러/미디어) 분석 모듈 |
| 사용자는 UI만 봄 | 갤러리 + YouTube = 사용자 인터페이스 |
| 내부는 복잡하지만 밖에서는 단순 | 뒤죽박죽看起来지만, 실제로는 연결되어 있음 |
| 부팅되면 OS는 스스로를 감춤 | 서버.py가 돌아가면 철학은 보이지 않게 작동함 |
| 버전업이 존재 | "과거의 나 MCP" vs "현재의 나" 카운터 = OS 업데이트 |
