## 3. 업로드 파이프라인 (PD 시스템)

### upload.cjs (`dtslib-papyrus/tools/youtube/upload.cjs`)
- 상태: ✅ 완성 (2월부터 사용 중)
- 큐 구조: `uploads/pending/` → `upload.cjs` → `uploads/done/`
- 기본값: `privacy: "private"` (Human Gate 구조)
- done/ 이력: 13개 업로드 완료 기록 있음

### 스펙 JSON 형식
```json
{
  "account": "b",
  "title": "영상 제목",
  "description": "설명",
  "tags": ["태그1"],
  "category_id": "22",
  "privacy": "private",
  "file": "/path/to/video.mp4",
  "thumbnail": "/path/to/thumb.jpg"
}
```

| API | quota |
|-----|:-----:|
| videos.insert | 1,600 |
| thumbnails.set | 50 |
