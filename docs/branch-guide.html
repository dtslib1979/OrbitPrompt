Branch Guide

OrbitPrompt 템플릿을 브랜치에서 사용하는 방법

1. 개요

OrbitPrompt는 방송용 칠판 템플릿을 제공하는 PD 스튜디오입니다. 각 브랜치(KOOSY, GOHSY, LOTUS 등)는 이 템플릿을 가져와 자신의 브랜딩으로 커스터마이징할 수 있습니다.

2. 빠른 시작

1

템플릿 URL에 branch 파라미터 추가

가장 간단한 방법: URL에 ?branch=브랜치명 추가

https://dtslib1979.github.io/OrbitPrompt/prompts/broadcast/math-tutor.html?branch=koosy
2

또는 커스텀 스크립트 사용

더 세밀한 커스터마이징이 필요하면:

<script src="https://dtslib1979.github.io/OrbitPrompt/api/customize.js"></script>
<script>
  OrbitCustomize.init({
    branch: 'koosy',
    colors: {
      primary: '#ff6b6b',
      accent: '#ffd93d',
      bg: '#1a1a2e'
    },
    logo: 'https://koosy.kr/logo.png'
  });
</script>

3. 사용 가능한 템플릿

템플릿 포맷 용도
math-tutor Khan Academy 수학 과외 방송
music-curation NPR Tiny Desk 음악 큐레이션
memorial-tribute Ken Burns 추모 영상
pwa-demo Apple Keynote 앱 데모

4. 커스터마이징 옵션

색상

변수 설명 기본값
primary 주 브랜드 색상 #58a6ff
accent 강조 색상 #a371f7
bg 배경 색상 #0d1117
surface 카드/패널 색상 #161b22
text 텍스트 색상 #e6edf3

기타 옵션

옵션 타입 설명
logo URL 브랜치 로고 이미지 URL
font string 커스텀 폰트 이름
watermark boolean "Powered by OrbitPrompt" 표시 여부

5. 등록된 브랜치 프리셋

브랜치 Primary Accent

6. API 엔드포인트

엔드포인트 설명
/api/templates.json 템플릿 카탈로그 (버전, URL, 설명)
/api/customize.js 브랜치 커스터마이징 스크립트

7. 브랜치 등록 요청

새 브랜치 프리셋 등록은 dtslib-branch 레포에서 요청하세요.
