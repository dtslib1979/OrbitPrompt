## 1. 핵심 진단

현재 OrbitPrompt는 철학, 구조, 구현, 증명은 이미 강하다. 문제는 기능이 부족한 것이 아니라 **첫 화면에서 제품 가치가 즉시 해석되지 않는 것**이다.

따라서 지금 필요한 업데이트는 기능 추가가 아니라, 아래 3가지를 한 번에 해결하는 **전파 레이어 패치**다.

1. 한 줄 가치 제안
2. 전문가용 입구(CTA)
3. STT→MCP 변환 서사의 시각화

## 3. 랜딩 페이지 구조 개편안

- 상단 라벨: Philosopher's Laboratory
- 메인 타이틀: 사고가 전시가 되는 순간
- 추가 가치 제안: 경력 10년+ 전문가의 판단을 MCP로 구조화합니다.
- 보조 서브카피: 당신의 20년 노하우를 말에서 시뮬레이터로 바꿉니다.

### 섹션 2 — Proof
- 실증 완료: Election MCP v2.0
- 10개 도구 · 12개 delta layer · direction_acc 100%

### 섹션 3 — For Experts
전문가를 위한 MCP 복제 워크숍

3개 카드:
- 제조·생산 관리자: 내 20년 현장 노하우를 MCP로
- 교육자·강사: 내 커리큘럼을 MCP로
- 법률·컨설팅: 내 판단 전략을 MCP로

### 섹션 4 — Voice to MCP
말이 시스템이 되는 과정

1. 전문가가 자기 경험을 말한다.
2. STT가 발화를 기록한다.
3. LLM이 규칙과 구조를 해석한다.
4. MCP가 실행 가능한 판단 엔진으로 고정한다.
