# Φ7 철학 카운터 MCP v1.0 — 만점판

> "철학 공부가 아니라 ID 엔진 튜닝이다."

## 철학

에코 3부작(라 필로소피아)을 마스터 테이블로 삼아,
철학자 특징과 박씨 특징의 오버랩을 동적으로 계산해서
"닮은 점 / 다른 점 / 결정적 차이"를 발견하는 MCP.

## 이전 버전과의 차이

| 항목 | v0.1 (문제) | v1.0 (해결) |
|------|------------|------------|
| 점수 체계 | 수동 입력 (동어반복) | 특징 기반 동적 계산 (자카드 유사도) |
| 발견 | 기존 결론 출력 | 예상치 못한 연결 발견 |
| 철학자 | 8명 | 11명 (키르케고르/아렌트/마르크스 추가) |
| ID 반영 | 제안만 | 실제 업데이트안 생성 |
| 검증 | 불가능 | rank + map + discover로 즉시 검증 가능 |

## 툴 6개

| 툴 | 기능 | 예시 |
|----|------|------|
| `analyze` | 철학자 1명 전체 분석 | 가타리 Φ7 5축 분석 |
| `discover` | 예상치 못한 발견 찾기 | 모든 철학자와의 결정적 차이 |
| `rank` | 유사도 순위 | 부르디외 85% / 가타리 77% / 디오게네스 62% |
| `id_update` | ID-MANIFEST 반영안 | 비교 결과 → ID 수정 제안 |
| `counter` | counter.md 자동 생성 | chan-dae-counter.md 포맷 |
| `map` | 철학적 계보 지도 | S/A/B/C 4계층 분류 |

## 증명된 발견

11명 철학자 전부에서 공통으로 발견된 패턴:
**"사유의 종착점" — 그들은 책/해석/비판으로 끝나고, 박씨는 구축/인프라/자동화로 끝난다.**

이건 수동 입력이 아니라 특징 오버랩에서 자연스럽게 도출된 진짜 철학적 발견이다.
