🧮 F=ma 정치 변화력 모델

# 결정론적 수학 — 동일 입력 = 동일 출력 (LLM 개입 0)

m_eff = m × k
a = 0.30·g + 0.30·d + 0.20·i + 0.20·o
F = m_eff × a

# 전복조건: F > R → 전복가능 / F ≤ R → 체제유지

# 변수 정의 (0~10):
g = 계급격차    d = 정치박탈감
i = 정체성공명도    o = 조직화능력
m = 인구질량(0~1)    k = 미디어계수(0~1)
R = 체제저항력

⚔️ 도전자 130% 법칙

🧠 도덕 서사 (김어준식 정신승리)
"우리가 열심히 안 해서 졌다"

→ 구조를 인정하지 않고 노력 부족으로 환원.
→ 다음에도 똑같은 구조에서 질 확률 높음.
⚡ 물리 서사 (MCP 구조 해석)
"이 구조에서는 130%를 때려넣어야 겨우 승부가 열린다"

→ 챔피언은 o(조직력)+R(체제저항)이 기본 30% 우위.
→ 도전자는 g/d/i/o 합계가 30% 더 높아야 F가 같아짐.
→ 100%로는 절대 못 이김. 130%가 기본.
🔑 핵심: "우리가 열심히 안 해서 졌다"는 말은 사실 "도전자 마인드셋이 부족했다"는 뜻이다.
도전자는 챔피언보다 30% 더 높은 g/d/i/o를 만들어야 겨우 승부를 걸 수 있다. 100%로는 안 된다.

🏟️ 지역 구조적 불가능 — 김부겸 케이스

📍 TK (대구·경북)
김부겸 — 대구 출신 유스 선수, 개인 능력치는 최상위.

하지만: 몇 번을 도전해도 안 됐다.

원인: regional_bossism=9, organization_power=8, R=9
→ 구조적으로 홈팀이 정해진 리그.
→ F/R < 0.5 = '구조적 불가능' 구간.
→ 개인 능력치 최대 발휘해도 승부 불가.
📍 호남 (광주·전라)
보수 후보 전반 — 전라도에서 보수가 안 되는 이유.

TK와 동일한 구조: R=9, o=7, regional_bossism=8.
→ 프레임 자체가 다름. '이긴다'의 정의가 다른 지역.
→ 도전자는 여기서 '승리'를 목표로 하면 안 되고,
   '구조를 조금씩 깎는 역할'로 접근해야 함.
🔑 MCP 해석: 김부겸 사례는 "개인 능력치 vs 지역 구조"의 대표 케이스다.
hero_score(개인)는 high지만, regional_bossism + far_right_drift + organization_power가 R을 극한으로 높여
calc_political_force 결과가 항상 '체제유지'로 고정되는 지역.

이걸 '열심히 안 해서'로 설명하면 김어준식 정신승리가 되고,
'구조적 불가능 구간'으로 인정하면 박씨식 물리 서사가 된다.

🧩 델타 레이어 — 제약 조건 & 실수 가능성

이상적 모델(g/d/i/o/R → F=ma) 위에 실제 선거 결과에서 추출한 제약 조건을 얹은 레이어

🔴 hard_constraint
지역 구조적 불가능
F/R < 0.5 = 구조적 불가능 구간. 개인 능력치 무력화.
TK(김부겸) · 호남(보수후보) — 홈팀이 정해진 리그.
🟡 soft_constraint
도전자 130% 법칙
챔피언 o+R 30% 우위. 도전자 g/d/i/o 합계 30% 높아야 F 같음. 100%로는 절대 못 이김.
🟡 structural_penalty
진영 분열 패널티
elite_fragmentation > 7 → challenger_force -30%. 평택=진영분열로 패배.
🟢 structural_bonus
현직 미디어 보너스
k_media +0.10, R +0.5. 서울(오세훈 역전승) · 부산(국힘 조직력).
🔵 signal
서울 바로미터
서울시장 선거 = 전국 판세 m(질량). 서울 승리 진영이 판세 주도. m_factor = 1.3.
🟣 swing_factor
실용 블록 유동성
pragmatic(이재명) 블록 ±15%p 변동 가능. 3블록 중 가장 불안정.
🔑 MCP 10번째 도구: delta_layer_info() — 이 델타 레이어 전체를 JSON으로 반환.
모델 구조: ideal core (F=ma) + delta constraints (6개 레이어)
