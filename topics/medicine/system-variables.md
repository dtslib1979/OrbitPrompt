# 시스템 변수화 — 물질과 비물질

> 몸의 모든 시스템을 **물질(Material) / 비물질(Immaterial)** 로 나누고,
> 각각을 변수(Unit/Variable)로 추상화하면 상위 카테고리가 보인다.

---

## 1. 분류 기준

| 구분 | 특징 | 전달 매체 | 속도 | 예 |
|------|------|-----------|------|----|
| **비물질 (Immaterial)** | 신호·정보·제어 | 전기·화학 신호 | 빠름 (ms~s) | 신경, 호르몬, 면역기억 |
| **물질 (Material)** | 자원·에너지·구조 | 유체·고체 | 느림 (s~min) | 혈액, 산소, 영양분, 뼈 |

비물질이 물질을 제어하고, 물질이 비물질을 실어 나른다.

---

## 2. 비물질 계열 — 신호와 제어

### 2.1 신경계 `S(t)` — Signal, Communication

```
변수:     S(t) = 전기화학적 신호 (mV·Hz)
입력:     감각 수용기 (sensory input)
처리:     뉴런 네트워크 (integration + threshold)
출력:     운동 명령 / 호르몬 분비 지시
피드백:   반사궁 (순간) / 장기기억 (영구)
```

| 속성 | 값 |
|------|-----|
| 매체 | 전기 (활동전위) + 신경전달물질 (화학) |
| IT 대응 | CPU·버스·인터럽트·이벤트 루프 |
| 국가 대응 | 대통령·국회·지휘통신체계 |
| 비유 | **전화 통화** — 메시지를 전달하고 명령을 내린다 |

### 2.2 내분비계 `H(t)` — Hormonal Setpoint, Regulation

```
변수:     H(t) = 호르몬 농도 벡터 (pg/mL·n개 차원)
입력:     혈중 농도·스트레스·일주기 리듬
처리:     시상하부-뇌하수체-표적기관 축 (HPT, HPA, HPG)
출력:     대사율·성장·스트레스 반응·생식 리듬 조절
특징:     신경계보다 느리지만 지속적, 전체 시스템의 "설정값"
```

| 속성 | 값 |
|------|-----|
| 매체 | 혈액 (화학 신호) |
| IT 대응 | 컨피그 파일·정책 엔진·스케줄러·QoS |
| 국가 대응 | 기재부·한은 (금리·세율) |
| 비유 | **온도조절기·타이머** — 시스템 전체 파라미터를 튜닝 |

### 2.3 면역계 `I(t)` — Immunity State, Pattern Recognition

```
변수:     I(t) = 면역 상태 벡터 (항체가·T세포 활성도·염증 수준)
입력:     항원 (pathogen pattern) / 손상 신호 (DAMPs)
처리:     선천면역 (즉시) → 적응면역 (학습·기억)
출력:     염증 반응·항체 생성·세포 제거
특징:     기억 (memory) = 재발 방지의 핵심
```

| 속성 | 값 |
|------|-----|
| 매체 | 세포 (백혈구) + 단백질 (항체·사이토카인) |
| IT 대응 | 방화벽·IDS/IPS·안티바이러스·비정상 탐지 |
| 국가 대응 | 국방부·질병관리청·경찰 |
| 비유 | **보안 경비 시스템** — 침입 탐지·차단·기억 |

## 3. 물질 계열 — 자원과 구조

### 3.1 순환계 `F(t)` — Flow, Logistics

```
변수:     F(t) = 혈류량 벡터 (mL/min·장기별)
방정식:   Q = ΔP / R  (유량 = 압력차 / 저항)
입력:     심장 박출량 · 혈관 직경 (저항)
처리:     산소·영양·호르몬·면역세포 운반
출력:     조직 관류 · 온도 조절 · 노폐물 회수
```

| 속성 | 값 |
|------|-----|
| 매체 | 혈액 (적혈구·백혈구·혈소판·혈장) |
| IT 대응 | 네트워크 스택·데이터버스·메시지 큐 |
| 국가 대응 | 재정·물류·교통·전력망 |
| 비유 | **택배·도로 시스템** — 피가 모든 걸 싣고 다닌다 |

### 3.2 호흡계 `O(t)` — Oxygen Exchange, Gas Interface

```
변수:     O(t) = 산소분압 (PaO₂·mmHg) / CO₂분압 (PaCO₂)
방정식:   V = (Vₜ - Vᴅ) × f  (분시환기량 = 1회환기량 × 호흡수)
입력:     대기 (21% O₂) → 폐포
처리:     확산 (가스 교환) · 헨리 법칙 (혈중 용해)
출력:     세포 호흡용 O₂ 공급 · CO₂ 배출
특징:     순환계와 직결 — 피가 산소를 운반하므로
```

| 속성 | 값 |
|------|-----|
| 매체 | 공기 (O₂, CO₂, N₂) |
| IT 대응 | 전원 공급·쿨링 시스템·리소스 인/아웃 |
| 국가 대응 | 환경부·에너지 수입 |
| 비유 | **환기·발전소** — 유닛(산소)이 피(혈액)로 들어간다 |
| 순환계와 차이 | "피"라는 운반체는 같은데, 싣는 짐이 **산소**로 바뀐 것뿐 |

### 3.3 소화계 `E(t)` — Energy Processing, Factory

```
변수:     E(t) = 에너지 플로우 (kcal/h · 영양소 분해율)
방정식:   E_net = E_intake − E_used − E_stored
입력:     음식 (탄수화물·지방·단백질·비타민·미네랄)
처리:     기계적 분해 (저작·연동) → 화학적 분해 (효소) → 흡수
출력:     혈중 영양소 (포도당·아미노산·지방산) → ATP 생산
특징:     **공장 + 경제 시스템** — 원자재 들여와 가공·분배·저장
```

| 속성 | 값 |
|------|-----|
| 매체 | 음식 → 유미즙 → 혈중 영양소 |
| IT 대응 | I/O 시스템·파서·ETL 파이프라인·데이터 가공 |
| 국가 대응 | 농식품부·산업부·무역 (공급망 전체) |
| 비유 | **공장·경제 시스템** — 원자재 → 가공품 → 저장·분배 |

### 3.4 비뇨계 `W(t)` — Waste Processing, Cleanup

```
변수:     W(t) = 노폐물 농도 (mg/dL · GFR mL/min)
방정식:   GFR = (U × V) / P  (사구체여과율)
입력:     혈액 (노폐물·독소·과잉 전해질)
처리:     사구체 여과 → 세뇨관 재흡수·분비 → 농축
출력:     요 (urea·creatinine·과잉수분)
특징:     **쓰레기 재처리 공장** — 태우거나 버리거나 재흡수하거나
```

| 속성 | 값 |
|------|-----|
| 매체 | 혈액 → 요 |
| IT 대응 | GC·로그 로테이션·캐시 정리·리소스 해제 |
| 국가 대응 | 환경부·폐기물 처리·금융 부실 정리 |
| 비유 | **재활용·하수 처리장** — 못 쓰는 거 정리해서 배출 |

### 3.5 근골격계 `M(t)` — Mechanical Structure, Motion

```
변수:     M(t) = 위치·자세·힘 (m·N·degrees)
방정식:   τ = r × F  (토크 = 팔길이 × 힘)
입력:     신경 명령 (운동신경)
처리:     근수축 (actin-myosin crossbridge) → 관절 운동
출력:     물리적 움직임·자세 유지·힘 발생
특징:     몸의 물리적 프레임 — 하드웨어 섀시
```

| 속성 | 값 |
|------|-----|
| 매체 | 근육·뼈·인대·힘줄 |
| IT 대응 | 하드웨어 섀시·로봇 암·액추에이터 |
| 국가 대응 | 국토부·SOC·도로·철도·건물 |
| 비유 | **건물 골조·로봇 프레임** |

### 3.6 피부/외피계 `B(t)` — Boundary, Interface

```
변수:     B(t) = 경계 상태 (체온·pH·수분 증발량)
입력:     외부 환경 (온도·습도·UV·병원체)
처리:     각질층 barrier → 멜라닌 방어 → 땀 배출
출력:     체온 조절 · 물리적 차단 · 감각 · 비타민 D 합성
특징:     시스템과 환경의 경계 — **UI/UX 레이어**
```

| 속성 | 값 |
|------|-----|
| 매체 | 피부·털·손톱·땀샘 |
| IT 대응 | 케이스·UI/UX·물리 보안·외부 인터페이스 |
| 국가 대응 | 외교부·관세청·출입국·문화·홍보 |
| 비유 | **국경·외관·UI** — 밖과 안을 나누는 막 |

### 3.7 생식계 `R(t)` — Reproduction, Deployment

```
변수:     R(t) = 생식 상태 (성호르몬 농도·배란/정자 생성)
입력:     유전 정보 (DNA) + 영양 + 환경 신호
처리:     감수분열 → 생식세포 형성 → 수정 → 발생
출력:     새로운 개체 (복제+변이)
특징:     시스템이 자기 자신을 **복제·배포**하는 유일한 기능
```

| 속성 | 값 |
|------|-----|
| 매체 | 생식세포 (정자·난자) |
| IT 대응 | 배포 파이프라인·프로비저닝·CI/CD·버전 관리 |
| 국가 대응 | 복지부·교육부·중소벤처부 (인구·교육·창업) |
| 비유 | **신규 인스턴스 생성·다음 세대 배포** |

## 4. 상위 카테고리 — 2×6 매트릭스

| | 비물질 (Immaterial) | 물질 (Material) |
|--|-------------------|-----------------|
| **제어/정보** | 신경계 `S(t)` | — |
| **설정/조율** | 내분비계 `H(t)` | — |
| **방어/보안** | 면역계 `I(t)` | 피부계 `B(t)` |
| **에너지/자원** | — | 소화계 `E(t)` · 호흡계 `O(t)` |
| **물류/수송** | — | 순환계 `F(t)` |
| **정화/정리** | — | 비뇨계 `W(t)` |
| **구조/운동** | — | 근골격계 `M(t)` |
| **재생/복제** | — | 생식계 `R(t)` |

## 5. 이 모델의 핵심 — 유닛 변수로 보면

모든 시스템은 결국 **하나의 변수**로 추상화된다:

```
System(t) = f(input, state, feedback)
```

| 계열 | 공통 속성 |
|------|-----------|
| **비물질** | signal → integrate → decide → output |
| **물질 (Resource)** | import → transform → transport → use → export |
| **물질 (Structure)** | build → maintain → repair → adapt |

신경계와 순환계의 차이는:
- 신경계: **신호**만 주고받음 (비물질)
- 순환계: **피**라는 매체에 모든 걸 실어나름 (물질)

피와 산소의 차이는:
- 순환계(F) = 운반체 (택배 차량)
- 호흡계(O) = 운반물 (그중 산소라는 특수 유닛)

**같은 운반 시스템이지만 "무엇을 싣느냐"가 시스템을 분류한다.**

## 6. 국가·IT 대응 한 줄 요약

| 변수 | 인체 | IT | 국가 |
|------|------|----|------|
| `S(t)` | 신경계 | CPU·OS·네트워크 | 청와대·국회 |
| `H(t)` | 내분비계 | 컨피그·정책 엔진 | 기재부·한은 |
| `I(t)` | 면역계 | 방화벽·보안 | 국방·방역 |
| `B(t)` | 피부계 | UI·케이스 | 외교·국경 |
| `F(t)` | 순환계 | 데이터버스·네트워크 | 물류·재정 |
| `O(t)` | 호흡계 | 전원·쿨링 | 에너지·환경 |
| `E(t)` | 소화계 | ETL·파서·I/O | 산업·공급망 |
| `W(t)` | 비뇨계 | GC·로그 정리 | 폐기물·규제 |
| `M(t)` | 근골격계 | 하드웨어·액추에이터 | SOC·건설 |
| `R(t)` | 생식계 | CI/CD·프로비저닝 | 교육·출산·창업 |

## 7. 학습 메타 — 이 프레임이 외우기 쉬운 이유

> "새로 외우는 게 아니라, 기존 모델에 태그만 붙인다."

### 일반적인 학습 (비효율)

```
생물학 용어 목록 ✕ 의학 용어 목록 ✕ CS 용어 목록 ✕ 경제 용어 목록
= 4배로 따로따로 기억해야 함
```

### 이 프레임의 학습 (효율)

```
하나의 시스템 구조 (signal → process → output)
    ├── 태그: 신경계 (인체)
    ├── 태그: CPU·OS (IT)
    └── 태그: 청와대·국회 (국가)
= 기존 직관 위에 태그만 추가
```

### 왜 효과적인가

| 전통적 방식 | 이 프레임 |
|-----------|----------|
| "신경계는 뉴런·시냅스·활동전위…" | "신경계 = 전화 통화, CPU랑 똑같네" |
| "순환계는 심장·혈관·혈액…" | "순환계 = 택배, 네트워크랑 똑같네" |
| "소화계는 위·장·간·췌장…" | "소화계 = 공장, ETL이랑 똑같네" |
| 각각 암기 필요 | **이미 아는 구조에 덧씌우기만 하면 됨** |

### 앞으로 새로운 정보가 들어와도

1. "이건 어떤 시스템 레이어를 건드리는가?"
2. "물질 계열인가, 비물질 계열인가?"
3. "IT/국가에서는 뭐에 해당하는가?"

이 세 질문만 하면 자동으로 머릿속 맵에 들어온다.
**공부량은 줄고, 연결성은 늘고, 기억 유지 시간은 길어진다.**
