# 몸-시스템 모델 실전 사용법 — 5가지 장면

> 몸이라는 컨닝페이퍼를 평소에 어떻게 꺼내 쓸 것인가.

---

## 1. 아침 30초 바디 스캔

몸 상태를 5축으로 훑는다. 5초씩만.

- **신경(제어)** — 머리 맑은가? 두통·어지럼?
- **순환(자원)** — 손발 차가운가?
- **면역(방어)** — 목·코·피부 어딘가? 염증 신호?
- **소화+호흡(에너지)** — 속 더부룩? 숨 깊게 들어오나?
- **내분비(설정)** — 스트레스 레벨? 턱·어깨 긴장?

→ 축 하나라도 이상하면, body-solution.md의 4변수(수면/스트레스/혈당/영양)를 점검한다.

---

## 2. 새로운 정보를 몸에 태깅한다

뉴스·책·대화에서 모르는 개념이 나오면 즉시 질문: "이건 몸의 어떤 계통이지?"

| 개념 | 태깅 | 이유 |
|------|------|------|
| GDP 성장률 | 순환계(피) | 돈의 흐름 |
| 방화벽 | 면역계 | 외부 침입 차단 |
| 이사 바꾸자 | 내분비계 | 설정/방향 변경 |
| 서버 터짐 | 신경계 | 정보 처리 중단 |
| 연봉 협상 | 순환계 | 자원 분배 |
| 회사 비전 | 내분비계 | 방향 설정 |

외우려 하지 말고 태깅만 하면 몸 그림 위에 자동 정리된다.

---

## 3. 무언가 만들 때 몸을 컨닝한다

팀짜기·서비스 기획·규칙 만들 때 5개 질문:

1. **신경(통신)** — 누가 의사결정하고 정보는 어떻게 오가나?
2. **순환(자원)** — 돈·데이터·물자는 어떻게 흐르나?
3. **면역(방어)** — 리스크는 누가 감시하고 문제 터지면 누가 막나?
4. **호흡+소화(에너지)** — 투입→처리→산출 구조는?
5. **내분비(설정)** — 전체 분위기·규칙·리듬은 누가 바꾸나?

몸이 이미 5개 다 돌리고 있으니까, 그 구조를 그대로 갖다 쓴다.

## 4. 문제 터졌을 때 "국소 vs HQ" 판단

문제 생기면 1초 분류:

| 국소 (정비소) | HQ (본부) |
|-------------|-----------|
| 손가락 베임 | 2주째 잠 안 옴 |
| 차 타이어 펑크 | 엔진 경고등 |
| 직원 1명 실수 | 팀 전체 사기 저하 |
| 감기 | 자가면역·만성 염증 |

**국소** → body-unit-dictionary.md의 정비소 목록에서 해당 해법
**HQ** → body-solution.md의 4변수(수면/스트레스/혈당/영양) 점검

## 5. AI랑 일할 때 역할 분담

| 업무 | 처리 주체 | 몸 비유 |
|------|----------|---------|
| 반복 코드·문서 | AI | 자율신경 (자동) |
| 큰 그림·방향 | 나 | 전두엽 (의식적 결정) |
| 이상 감지·알림 | AI 감시 + 내 판단 | 면역계가 보내는 신호 |
| 새로운 거 배우기 | 내가 분류하고 AI가 정리 | 감각 입력 → 피질 처리 |

이거 하나만 기억해도 AI랑 같이 일할 때 "뭘 맡기고 뭘 할지"가 3초에 결정된다.
