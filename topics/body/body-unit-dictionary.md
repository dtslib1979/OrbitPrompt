# 몸 유닛 사전 — 초등학생 버전

> 몸을 10개 재료 + 5개 시설 + 3개 모드로 분해했다.
> 의학 공부할 때 이 표만 머리에 넣고 다녀라.

---

## PART 1: 몸을 구성하는 10가지 재료

| # | 유닛 | 진짜 정체 | 느낌 | 하는 일 |
|---|------|-----------|------|---------|
| 1 | **물** | 혈액(혈장) + 조직액 | 정수기 물 | 영양·산소·약 전부 이 물에 태워서 온몸으로 보냄. 세포는 이 물 속에서 산다 |
| 2 | **기름물** | 림프액 | 개수대 기름 낀 물 | 물+기름+찌꺼기 섞인 정화용 물. 조직에서 더러워진 물을 빼내서 림프절(정수기)로 보내고, 거기서 면역세포가 "이상 없음" 검사함 |
| 3 | **돌** | 뼈 | 콘크리트 기둥 | 몸의 프레임. 속에 있는 골수는 피공장(혈액을 만듦) |
| 4 | **고무줄** | 근육 | 고무줄 수천 개 | 당기면 수축, 놓으면 이완. 돌(뼈)을 당겨서 몸을 움직임 |
| 5 | **공기** | 산소(O₂) | 불 때는 산소 | 피 속의 물에 녹아서 세포로 감. 세포는 이 산소로 불(에너지)을 때서 ATP를 만듦 |
| 6 | **연기** | 이산화탄소(CO₂) + 노폐물 | 공장 굴뚝 연기 | 세포가 불 땐 후 남은 배기가스. 피에 녹아서 폐로 가서 내뱉음 |
| 7 | **전기** | 신경 활동전위 | 전기줄 타고 가는 전기 | 느낌(더워, 아파)을 전기로 바꿔서 뇌로 보내거나, 뇌에서 "손 들어" 명령을 전기로 내려보냄 |
| 8 | **방송** | 호르몬 | 학교 전체방송 / 뉴스레터 | "혈당 올려라", "성장해라", "잘 준비해라" 같은 전역 설정을 피(물)에 띄워서 전송 |
| 9 | **무전기** | 사이토카인 | 경비원 무전기 | 면역세포끼리 "여기 적 있다", "도와줘" 하는 국소 신호. 근처 세포들만 듣는다 |
| 10 | **배터리** | ATP (세포 에너지) | 충전지 / 건전지 | 세포가 쓰는 유일한 에너지. 음식→ATP 충전, ATP→에너지 방전. 몸의 통화(currency) |

## PART 2: 처리 시설 — 몸 속 공장 7곳

| # | 시설 | 진짜 정체 | 느낌 | 하는 일 |
|---|------|-----------|------|---------|
| 1 | **펌프** | 심장 | 자동 물펌프 | 1분에 60~100번 펌핑해서 물(혈액)을 온몸으로 보냄. 멈추면 3분 안에 사망 |
| 2 | **풀무** | 폐 | 대장간 풀무 / 송풍기 | 공기를 들이마셔서 산소를 물(혈액)에 녹이고, 물에서 연기(CO₂)를 빼내서 내뱉음 |
| 3 | **분쇄기** | 위 + 창자 | 고기 분쇄기 + 추출기 | 음식을 갈아서 영양소를 뽑아내고, 물(혈액)에 태워서 보냄. 못 쓰는 건 똥으로 만듦 |
| 4 | **화학 공장** | 간 | 정유 공장 + 필터 | 분쇄기에서 온 원료를 가공(포도당 저장·해독·단백질 합성). 몸 안의 만능 화학 공장 |
| 5 | **정수 필터** | 신장 | 정수기 필터 | 물(혈액)을 걸러서 찌꺼기(요소·노폐물)를 빼내고 오줌으로 버림. 물의 질 유지 |
| 6 | **경비 초소** | 림프절 + 비장 | 경비실 + 검문소 | 기름물(림프)이 지나갈 때 "이상 없음" 검사. 적(바이러스·세균) 발견 시 면역 공격 개시 |
| 7 | **벽 + 대문** | 피부 + 점막 | 집 벽 + 현관문 | 몸과 바깥의 경계. 물리적 방어 + 체온 유지 + 감각. 뚫리면 면역계 비상 발동 |

## PART 3: 몸의 모드 3가지

| # | 모드 | 느낌 | 하는 일 | 언제? |
|---|------|------|---------|-------|
| 1 | **밤 모드** (수면) | 불 끄고 청소기 돌림 | 뇌청소(glymphatic), 세포청소(오토파지), 면역정비, 호르몬 재설정 | 매일 밤 7~8시간 |
| 2 | **비상 모드** (스트레스) | 불 켜고 전투 준비 | 교감신발동 → 혈압↑·혈당↑·소화정지·면역일시억제. 적과 싸우거나 도망치는 모드 | 위험 상황, 단기 OK |
| 3 | **정비 모드** (휴식·소화) | 소파에 누워 쉼 | 부교감신경 활성화 → 소화·회복·면역정비. 비상 모드의 반대 | 식후, 안전할 때 |

## PART 5: 이걸로 의학 읽는 법

의학 정보가 들어오면 아래 순서로 분류한다:

**Step 1:** "이게 몸의 어떤 재료, 시설, 모드를 건드리는가?"

| 의학 용어 | 유닛으로 보면 |
|-----------|-------------|
| 빈혈 | **물(혈액)** 속 적혈구 부족 = 택배 기사 부족 |
| 림프절 부음 | **경비 초소**에서 전투 중 = 적과 싸우는 중 |
| 골절 | **돌(뼈)** 금감 |
| 근육통 | **고무줄(근육)** 손상 or 피로 |
| 고혈압 | **펌프(심장)**가 너무 센데 **물길(혈관)**이 좁아짐 |
| 당뇨 | **방송(인슐린)**이 안 오거나, 세포가 방송을 무시함 |
| 염증 | **무전기(사이토카인)**가 계속 울려서 **비상 모드**가 길어짐 |
| 스트레스 만성 | **비상 모드**를 하루 종일 켜놓음 → 소화·면역·재생 다 망가짐 |
| 수면 부족 | **밤 모드**를 안 킴 → 뇌청소X, 세포수리X, 면역정비X |
| 헤르페스 재발 | **면역 경비**가 약해져서(밤 모드 부족 or 비상 모드 만성) 바이러스가 기회를 탐 |
| 간 수치 이상 | **화학 공장(간)**이 과부하 걸림 |
| 신장 수치 이상 | **정수 필터(신장)**가 막히기 시작 |

**Step 2:** "이게 물질(재료·시설) 문제인가, 신호(전기·방송·무전기) 문제인가, 모드 문제인가?"

- **물질 문제** → 영양 보충, 수술, 약물
- **신호 문제** → 호르몬 치료, 신경 차단, 항염증
- **모드 문제** → 수면·스트레스 관리가 1순위

**Step 3:** "HQ에서 고칠 수 있는가, 국소 정비소로 보낼 것인가?"

- HQ 변수 (수면·스트레스·혈당·영양) → 직접 관리
- 국소 문제 (골절·상처·급성 감염) → 병원

## 1줄 요약

> 몸은 **물·기름물·돌·고무줄·공기·연기·전기·방송·무전기·배터리** 10개 재료,
> **펌프·풀무·분쇄기·화학공장·정수필터·경비초소·벽** 7개 시설,
> **밤모드·비상모드·정비모드** 3개 모드로 돌아간다.
>
> HQ(수면·스트레스·혈당·영양)만 잡아도 80%는 해결된다.

## PART 6: 하이브리드 자동차로 보는 몸

> 사람 = 하이브리드 자동차 (내연기관 + 전기모터).
> 소화·호흡 = 내연기관 / 신경·전기신호 = EV 모터.

### 인체 ↔ 하이브리드 차량 매핑

| 인체 | 자동차 부품 | 설명 |
|------|------------|------|
| **소화계 (위·장·간)** | 연료 탱크 + 연료 정제 계통 | 음식(원유) → 포도당·지방(정제 연료) |
| **호흡계 (폐)** | 공기 흡입구 + 에어 필터 | 엔진 연소용 산소 공급 |
| **미토콘드리아** | 엔진 실린더 | 연료+산소 → 폭발 → ATP(에너지) |
| **심장** | 연료 펌프 + 워터 펌프 | 1분 60~100회 펌핑 |
| **혈액** | 연료 + 냉각수 + 엔진오일 혼합액 | 산소·영양·호르몬·면역세포 전부 이 액체에 타고 다님 |
| **혈관** | 연료 호스 + 냉각수 배관 | 전신에 깔린 배관망 |
| **림프계** | 엔진오일 회수 + 필터 시스템 | 더러워진 오일(림프) 모아서 필터(림프절) 통과 후 재사용 |
| **신경계** | ECU(엔진제어유닛) + 전기 배선 | 센서 입력 → 처리 → 명령 출력 |
| **뇌** | 자율주행 컴퓨터 + 운전자 | 최종 의사결정. 가고 멈추고, 모드 선택 |
| **호르몬** | 드라이브 모드 (스포츠/에코/컴포트) | "스포츠 모드=교감↑" / "에코 모드=부교감↑" / "컴포트=항상성" |
| **뼈** | 섀시 + 프레임 | 몸의 구조적 골격 |
| **근육** | 액추에이터 + 모터 + 구동축 | 전기/유압 신호 받아서 실제 움직임 생성 |
| **관절** | 볼 조인트 + 서스펜션 | 프레임 연결 + 충격 흡수 |
| **피부** | 차체 + 도장 + 유리창 | 외부와의 경계. 방수·방풍·온도조절 |
| **면역계** | ABS·ESC·에어백·도난방지 | 이상 감지 → 차단 → 복구 |
| **신장** | DPF(매연필터) + 정화 시스템 | 혈액(배기가스) 걸러서 노폐물(오줌) 배출 |
| **간** | 촉매 변환기 + 오일 필터 | 독소 분해·영양소 저장·화학 처리 |
| **수면** | 주차 중 자동 충전 + 정비 모드 | 고전압 배터리 충전 + DPF 재생 + 시스템 점검 |
| **스트레스 반응** | 풀 악셀 + 레브 리미터 | 최대 출력 모드. 단기 OK, 장기면 엔진 과열 |
| **통증** | 대시보드 경고등 | "어딘가 고장 났음. 점검 필요" |

### 하이브리드 관점에서 보는 질환

| 증상 | 차로 보면 | 실제 문제 |
|------|----------|---------|
| 피로·무기력 | 배터리 방전 + 연료 부족 | ATP 부족, 수면 부족, 영양 불균형 |
| 혈압↑ | 연료 압력 과다 + 펌프 과부하 | 혈관 저항↑, 교감신경 항진 |
| 당뇨 | 연료 분사가 안 됨 (인젝터 고장) | 인슐린 저항 or 부족 |
| 염증 | 엔진 오버히트 + 경고등 | 면역계 과반응 |
| 헤르페스 재발 | 보안 시스템 알람 꺼짐 | 면역 감시 약화 |
| 골절 | 프레임 균열 | 뼈 구조 손상 |
| 근육통 | 구동계 마모 or 오일 부족 | 근육 미세 손상 or 회불충분 |
| 변비 | 배기가스 역류 | 장 운동 저하 |
| 불면 | 주차 중에도 시동 걸려 있음 | 수면 모드 진입 실패 |

### 그래서 이 매핑이 주는 것

> **몸 1개만 이해하면, 공짜로 4개 도메인이 따라온다.**

```
몸 (인체)
 ├── CS/IT (신경=CPU, 순환=네트워크, 면역=보안)
 ├── 국가 시스템 (HQ=대통령, 순환=재정·물류, 면역=국방)
 ├── 기계공학/메카닉 (소화=내연기관, 신경=ECU, 뼈=섀시)
 └── 전기차/하이브리드 (내연기관+전기모터=몸의 이중 에너지 시스템)
```

하나의 시스템 언어(재료·신호·처리시설·모드)로 네 도메인을 동시에 해석할 수 있다.
이게 이 프레임의 진짜 가치다.
